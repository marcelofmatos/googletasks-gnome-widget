# Termos de Serviço — Google Tasks Widget

**Última atualização:** 26 de abril de 2026

Estes Termos de Serviço ("Termos") regem o uso do aplicativo
**Google Tasks Widget** ("o Aplicativo"), um widget de desktop open source
para GNOME que exibe tarefas do Google Tasks.

Ao instalar, executar ou utilizar o Aplicativo, você concorda com estes
Termos. Se não concordar, não utilize o Aplicativo.

## 1. Sobre o Aplicativo

O Aplicativo é um software cliente que executa **localmente** no computador
do usuário. Ele utiliza a Google Tasks API, sob autorização explícita do
usuário via OAuth2, para exibir e gerenciar tarefas pessoais.

Não há servidor remoto operado pelos desenvolvedores. Toda comunicação de
rede ocorre diretamente entre o seu computador e os servidores do Google.

- Uma Conta Google ativa
